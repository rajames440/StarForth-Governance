# Governance Document Disposition Keywords
# Format: CATEGORY|keywords|escalate|pm_required|hold_if_found|route_to
# Pipeline parses this to make automatic disposition decisions
# escalate: true/false - escalate to PM immediately
# pm_required: true/false - require PM sign-off
# hold_if_found: true/false - hold document, don't route to signature yet
# route_to: comma-separated types to add as required signers
