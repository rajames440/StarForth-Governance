# ECR-001: Add Support for Floating Point Operations

## Description

Add floating point arithmetic support to the StarForth VM to enable scientific computing use cases.

## Impact

- Enables floating point calculations
- Requires new instruction set extensions
- Breaking change: new opcodes needed

## Implementation

1. Add FP stack to VM
2. Implement FP operators (+, -, *, /)
3. Add type checking
4. Update documentation
